# quick-sigma-server

## 0x01.简介

> quick-sigma-server是j2ee快启动框架，包含后台管理系统以及业务服务器的通用功能抽象。

## 0x02.技术选型

### - FrontEnd

1. Vue-Element

### - BackEnd

1. Spring Boot
2. Spring Security
3. Mybatis Plus
